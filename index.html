Universe of เกดสุรางค์

จักรวาลเกศสุรางค์ : มีเฟื้องจำกัด ต้องจ้างออเจ้าไหนมารีวิว

เป็นหนึ่งในโปรเจคต์การเรียนวิชา Social Network Analysis ฉบับ Non-IT โดยอ้างอิงความสัมพันธ์ของตัวละครจากชื่อดังเรื่อง บุพเพสันนิวาส และ พรหมลิขิต เพื่อวิเคราะห์ดูความสัมพันธ์ของแต่ละตัวละครแล้วที่มีความซับซ้อนข้ามภพข้ามชาติ ยังต้องการวิเคราะห์ในเชิงการตลาดเพื่อค้นหา Node ที่มีศักยภาพในการเป็น Influencerประจำกรุงศรีอยุธยานอกจากคุณหญิงการะเกด หรือ เกศสุรางค์นั่นเอง


Overview

จุดประสงค์ของโปรเจคต์นี้คือการหาความรูปแบบความสัมพันธ์ของกลุ่มสังคมที่รายล้อมตัวเกศสุรางค์เพื่อหา Influencer ด้วยโปรแกรม Gephi โดยใช้เทคนิคต่าง ๆ อาทิ

Centrality degree เสมือนเป็นหน่วยเรียกจำนวน edge ที่ node มี Node ไหนที่มี degree มากที่สุดจะเรียกได้ว่าเป็น node ที่มีความสัมพันธ์กับ node อื่นๆ
Clossness Centrality เป็นการค้นหา Node ที่เชื่อมโยงกับ Node อื่นได้รวดเร็ว หรือมีค่า edge ในระยะที่ 1 ช่วง เทียบได้กับค่าความสนิทชิดเชื้อ
Betweenness Centrality Node ที่มีบทบาทสำคัญในการเป็นสะพานเชื่อมต่อกับ node ในเครือข่ายที่สั้นที่สุด
Hub ทำตัวเหมือน Host ที่สามารถแพร่กระจายข่าวสารไปยัง Node ที่มีความสำคัญต่อเครือข่ายได้ดี
Authority Node ที่มีความสามารถเป็นแหล่งข่าวชั้นเลิศของเครือข่าย

Solution

นอกจากเกศสุรางค์จะเป็นทั้ง Center ทั้ง Between ของเรื่องราวแล้ว ด้วยการใช้โปรแกรม Gephi และอัลกอริทึ่ม HITS (Hyper Induced Topic Search) เห็นชัดว่า “พุดตาน” และ “พี่จ้อย” ทำหน้าที่เป็น Authority ที่น่าสนใจ กรณีนี้เราอาจเลือก พี่จ้อย ที่เป็นตัวละครรองลงมา เนื่องจากงบประมาณที่จำกัด ทั้ง พี่จ้อย ยังเป็นบุคคลที่ใคร ๆ ก็เข้าหา โดยเฉพาะเกศสุรางค์ ที่ทำหน้าที่เป็น HUB ผู้ประกาศกลางของหมู่บ้าน เรียกว่าหากแม่เกศเห็น พี่จ้อย หยิบจับอะไร มีโอกาสที่จะต้องถูกนำมาบอกต่อสูง
